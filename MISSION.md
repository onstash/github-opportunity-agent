# Mission

Build a first-principles Python agent runtime that teaches how agents work, then apply it to a practical read-only GitHub opportunity finder.

The learner should be able to understand:

- what an agent loop is
- how messages, state, results, and events fit together
- how streaming and final output coexist without being confused
- how to build toward a terminal-first, JSONL-backed runtime
- how to apply the same runtime ideas to finding open source contribution opportunities
