Pi CLI Skeleton Checklist

The smallest useful Pi AI CLI has five moving parts. If one is missing, the agent usually becomes hard to inspect or extend.

1. Models Collection

const models = builtinModels();
const model = models.getModel("openai", "gpt-4o-mini")!;

2. Typed Tools

const tools: Tool[] = [
  {
    name: "search_oss",
    description: "Find open source opportunities",
    parameters: Type.Object({ query: Type.String() })
  }
];

3. Context

const context: Context = {
  systemPrompt,
  messages,
  tools
};

4. Streaming Loop

const stream = models.stream(model, context);
for await (const event of stream) {
  // print text deltas, tool calls, and lifecycle events
}

5. Final Assembly

  • normalize raw tool outputs
  • rank opportunities
  • print short explanations
