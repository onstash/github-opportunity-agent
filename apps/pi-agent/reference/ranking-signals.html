Ranking Signals

A first ranker should be simple enough to inspect by hand. If you cannot explain why an item won, the scoring model is too opaque for this stage.

Core Signals

  • relevanceScore: fit to skills, interests, and target role
  • recencyScore: how fresh the opportunity is
  • effortScore: how approachable it looks right now
  • confidenceScore: how trustworthy the source appears

First Formula

total =
  relevanceScore * 0.45 +
  recencyScore   * 0.20 +
  effortScore    * 0.20 +
  confidenceScore * 0.15

Rule Of Thumb

Weight relevance highest first. A fresh but irrelevant opportunity should not outrank a slightly older, highly relevant one.

Explainability Pattern

{
  totalScore: number,
  reasons: string[]
}

Always keep a few reasons alongside the score.
