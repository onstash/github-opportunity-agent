# Mission

Build a working CLI-first TypeScript agent using `@earendil-works/pi-ai` that finds both:

- open source contribution opportunities on GitHub
- job opportunities related to GitHub projects, maintainers, and engineering teams

The learner should be able to understand:

- how Pi AI models, context, tools, and streaming events fit together
- how to structure a practical agent around source adapters, ranking, and explanations
- how to ship a first useful CLI before adding a UI
- how to keep the agent inspectable, testable, and easy to extend
- how to compare OSS and job opportunities in one ranked result set
